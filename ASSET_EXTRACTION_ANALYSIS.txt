== HOW TO EXTRACT GAME ASSETS ==

Step 1: Locate Game Asset Files
  Unity games typically store assets in:
  - [Game Install]/[GameName]_Data/
  - Android: /Android/obb/ or /Android/data/
  - Common files: level0, sharedassets0.assets, resources.assets

Step 2: Use AssetStudio (Recommended)
  1. Download AssetStudio from GitHub
  2. Run AssetStudioGUI.exe
  3. File → Load Folder → Select game's data folder
  4. Wait for assets to load
  5. Asset List tab shows all assets
  6. Select assets → Export → Export selected assets
  7. Choose output folder

Step 3: Organize Extracted Assets
  - Sprites will be exported as PNG files
  - Can filter by type (Sprite, Texture2D, AudioClip, etc.)
  - Can batch export all of one type

== ALTERNATIVE: Extract from APK (Android) ==

If game is Android-based:
1. Get APK file
2. Rename .apk to .zip
3. Extract contents
4. Find .obb files in /assets/
5. Use AssetStudio to open .obb files
6. Export assets
